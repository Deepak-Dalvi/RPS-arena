# RPS-arena (🎮 Rock Paper Scissors – RL Based Game)
Rock Paper Scissors game using Reinforcement Learning (Q-learning inspired logic)
This project is a web-based implementation of the classic Rock–Paper–Scissors game enhanced with basic Reinforcement Learning concepts. The game allows a user to play against an AI agent that improves its decisions over time based on previous outcomes.

---

## 🧠 Concepts Used
- Reinforcement Learning (basic idea)
- Exploration vs Exploitation
- Reward-based learning
- Adaptive decision making

---

## 🎮 How It Works
- Player selects Rock, Paper, or Scissors
- AI initially plays randomly
- AI updates its strategy based on wins/losses
- Over time, AI becomes more effective
